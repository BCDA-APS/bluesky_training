Test with Hello, World#

Test the new bluesky instrument with the quick Hello, World! code in the user/ directory.

Previously, a new bluesky directory was installed for this notebook from the template into temporary directory /tmp/bluesky.

Installation steps

Using commands in the Linux bash shell, remove any previous work:

$ /bin/rm -rf -rf /tmp/bluesky

Install from the template using new_bluesky_instrument.py:

$ python new_bluesky_instrument.py /tmp/bluesky
INFO:__main__:Requested installation to: '/tmp/bluesky'
INFO:__main__:Downloading 'https://github.com/BCDA-APS/bluesky_training/archive/refs/heads/main.zip'
INFO:__main__:Extracting content from '/tmp/bluesky_training-main.zip'
INFO:__main__:Installing to '/tmp/bluesky'

Start our session in the /tmp directory:

$ cd /tmp

At this point, you have assembled enough of the parts to test the initial installation with bluesky. Follow these steps to load the instrument package (in the /tmp/bluesky/instrument directory) for data collection activities. The name of each module is logged as it is loaded.

import pathlib, sys

path = str(pathlib.Path("/tmp") / "bluesky")
pathlib.os.chdir(path)
sys.path.append(path)
from instrument.collection import *
